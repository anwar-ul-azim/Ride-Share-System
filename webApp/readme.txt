Needed Resource Codes and Pre installs are here!

###############--pre_installation--####################

1. install node.js 
		
		download link (https://nodejs.org/en/)
			LTS version preferred.

2. install angular

		install using command line.
		 -> npm install -g @angular/cli

3. go to ["https://drive.google.com/open?id=1fOYihH9Fs-n1aO0m5FNSGIauOd9mgEyk" ]
 	and download the (node_modules.zip) file then extract here:
 		[...\cse482_group5_Ride-Share-System\webApp\rideshare\]
 	

 **if firebase,agm or other libraries are missing then install them:

	install firebase using the command:
		->npm install --save firebase

	install agm using the command:
		->npm install @agm/core --save
		->npm install --save agm-direction
		->npm install @types/googlemaps --save-dev

	install jquery using the command:
		->npm install jquery@1.9.1 --save

	install popper using the command:
		->npm install popper.js@1.12.9 --save

		->npm i bootstrap-social
		->npm i font-awesome
